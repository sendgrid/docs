--- layout: page weight: 0 title: Apps navigation: show: true --- {% anchor h2 %} List {% endanchor %}

List all of the available apps.

Note: The name entry is used in all the other API calls to identify a app.

Parameter Required Requirements Description
task Yes Must be set to getavailable Retrieve Available Apps
user Yes Customer must be registered under your account The customer who we will update
method Yes Must be set to apps Allows you to access apps functionality
{% apiexample get POST https://api.sendgrid.com/apiv2/reseller.manage api_user=your_sendgrid_username&api_key=your_sendgrid_password&task=getavailable&method=apps&user=customer@example.com %} {% response json %} { "name": "twitter", "title": "Twitter", "description": "This plugin allows you to send an email message to twitter", "activated": "false" } {% endresponse %} {% response xml %} twitter Twitter This plugin allows you to send an email message to twitter 0 ... {% endresponse %} {% endapiexample %}
