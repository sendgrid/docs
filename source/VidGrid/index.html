SendGrid's Technical Support Engineers love to help people, so we recorded a series of video tutorials called Project VidGrid. What would you like see added to VidGrid?

  • Event Notification Learn how to configure and set up an Event listener for the Event Notification App.
  • EventKit App Learn how to setup and use the EventKit; an open source app which uses our Event API.
  • Create a Subuser Learn how to create a subuser through the web interface.
  • Create a Subuser with its own whitelabel Learn what to create a subuser with its own whitelabeled domain.
  • Tools Learn how to use some of the helpful tools that we provide.
